Pre-Condition:
1. Hub onboarding successfully and in pairing mode
2. Install battery on the device
3. Long press Reset button to reset device
Device Onboarding Process:
1. Click Add Device button in SC App
2. Tap Reset button once to start pairing
3. In pairing process until indicator light turned off
4. Tap Reset button once to check the pairing result:
If indicator light fast blink once, it means pairing successfully;
Otherwise, it means failed
